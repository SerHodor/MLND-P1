# MLND-P1
## Create Env
```
	$	cd to folder
	$	conda create --name mlnd python=2
	$	activate mlnd
	$	conda install numpy matplotlib pandas jupyter notebook
	$	conda install scikit-learn=0.17
```
## Start Project
```
	$	cd to folder
	$	conda info --envs
	$	activate mlnd
	$	jupyter notebook boston_housing.ipynb
```
## Exit Project
```
	logout form notebook
	$	deactivate mlnd
```
